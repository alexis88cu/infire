# Atlas Web Agency — Lead Tracker

## How to Use
Copy this template into a Google Sheet or Airtable. Track every lead you contact.

---

## Lead Status Values
- `new` — found but not contacted yet
- `demo-sent` — demo was sent, waiting for reply
- `replied` — they responded
- `call-booked` — call scheduled
- `closed` — became a paying client
- `lost` — not interested
- `follow-up` — sent follow-up, waiting

---

## Google Sheets Columns

| Column | Description |
|--------|-------------|
| Business Name | Full business name |
| Industry | Roofing / HVAC / Restaurant / etc. |
| City | City in Florida |
| Website | Current website URL (or "none") |
| Phone | Business phone |
| Email | Business email |
| Instagram | @handle |
| Website Score | 1–10 (10 = no website) |
| Lead Score | 1–10 overall opportunity |
| Demo URL | demo.atlaswebagency.net/[slug] |
| Status | new / demo-sent / replied / closed / lost |
| Date Contacted | MM/DD/YYYY |
| Follow-Up Date | MM/DD/YYYY |
| Notes | Any context or conversation notes |
| Plan Sold | Starter / Professional / Premium |
| Setup Revenue | $250 or $500 |
| Monthly Revenue | $14.99 / $49 / $99 |
